# TauntTattle

Announces taunts and pulls in party or raid chat. Designed for World of Warcraft 1.12.1 and Turtle WoW.

## Features

- Announces successful and failed taunts (Resisted, Missed, Dodged, etc.).
- Supports Turtle WoW custom spells like Righteous Defense and Earthshaker Slam.
- Detects first-pulls in raids and logs them.
- Configurable output (Self-only, Party, or Raid).
- Minimal performance impact.

## Commands

- `/taunt` or `/taunttattle` - Toggle addon on/off.
- `/taunt self` - Toggle output to local chat only.
- `/taunt pulls` - Toggle pull detection (Raid only).
- `/taunt status` - Show current configuration.
